# CCA Portal

Full Stack CCA Licensing Portal built using React.js, Express.js and PostgreSQL.

## Features

- User Authentication
- Licensing Dashboard
- Intent Registration
- Secure APIs
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js
- PostgreSQL

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
node server.js
```
